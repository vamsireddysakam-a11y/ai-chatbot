# RULE-BASED AI CHATBOT

## Project Title

**Rule-Based AI Chatbot**

## Objective

The main goal of this project is to build a simple chatbot using Python. The chatbot can answer a few predefined questions and continue chatting with the user until the user decides to exit. It uses a dictionary to store responses and matches the user's input with those responses.

## Requirements

* Python 3.x
* Any Python editor such as VS Code, PyCharm, or IDLE

## Features

* Greets the user with messages like **"Hi"** and **"Hello"**.
* Answers simple questions such as:

  * How are you?
  * What is your name?
  * Who created you?
  * What can you do?
* Responds to greetings like **"Good Morning"** and **"Good Evening"**.
* Replies politely when the user says **"Thank you"**.
* Shows a help message with the available commands.
* Ends the conversation when the user types **"bye"**, **"exit"**, or **"quit"**.
* Gives a friendly message if it doesn't understand the user's question.
* Keeps running until the user chooses to exit.

## How to Run
